平等院

平等院を知る

Learn about BYODOIN

位於京都府宇治市的佛寺，沿著宇治 川邊興建，為日本早期木構建築，據信是 古代日本人對西方極樂世界的極致具體實現。

平等院最具代表性的建築 是面對阿字池而建，初期因置 奉阿彌陀如來與 52 尊雲中供養 菩薩像得名的阿彌陀堂，後因 阿彌陀堂外型似欲振翅而飛的 禽鳥，在中堂脊沿更有兩隻尊 貴象徵的金銅鳳凰像，遂在江 戶時期，更名為「鳳凰堂」。

宇治上神社

宇治上神社を知る

Learn about UJI GAMI JINJA

位於京都府宇治市宇治山田的神社。 毗鄰宇治神社。其本殿和拜殿都是日本國 寶，宇治上神社亦被列為世界文化遺產「古 都京都的文化財」之一。

宇治上神社位於宇治神社的北側，是 日本最為古老的神社，據考證創建於平安 時代，神社的本殿，是日本最古老的神社 建築，被列為日本的國寶。宇治上神社祭 奉應神天皇和倆位皇太子，宇治上神社內 還有一眼泉水，名為桐原水，是古傳宇治 七眼名水之一，也是現存唯一的一眼古名 泉，至今泉水仍源源不斷地噴湧。神社內 建於鐮倉時代的拜殿是日本重要文物。

貴船神社

貴傳神社を知る

Learn about KIFUNE JINJA

位在京都府京都市左京區的神社，貴 船神社為日本全國約 450 座貴船神社之總 本社，也是繪馬發祥之社。

關於貴船神社的由來，傳說貴船大神曾供奉於此。“ 為了國家和平，保護全體人民，在牛年的牛月之日， 從天降到貴船山的膝鏡岩中。” 在另一個傳說中，大約 1600 年前，神武天皇的母親玉賴姬命說：“我是玉賴公主。 而且，靈民的各種願望都會帶來好運。因此，在我們的船停靠的地方建造一座神社。” 它追溯了淀川、鴨川和貴船川，直到現在的奧宮作為水源。 據說貴船神社的起源是玉賴姬發現了神聖的淡水泉源吹井，在這裡建造了一座小神社。

伏見稻荷大社

伏見稻荷大社を知る

Learn about FUSHIMI INARI TAISHA

位於京都市伏見區內的神社，是遍及日 本全國各地約三萬所的稻荷神社之總本社， 也是近畿地方初詣參拜者最多的社寺。以境 內所擁有的「千本鳥居」聞名。

伏見稻荷大社主要是祀奉 以宇迦之御魂大神為首的諸位 稻荷神，自古以來就是農業與 商業的神明，除此之外也配祀 包括佐田彥大神、大宮能賣大 神、田中大神與四大神等其他 的神明。 由於每年都有大量的 香客前來神社祭拜求取農作豐收 、生意興隆、交通安全，使得該 神社成為京都地區香火最盛的 神社之一。

元離宮二條城

元離宮二条城を知る

Learn about NIJO-JO CASTLE

位於京都府京都市中京區二條城町的城 堡，始建於江戶時代初期慶長八年（1603 年）。 是江戶幕府第一代征夷大將軍德川家康作為守 護京都御所以及到京都拜見天皇住宿的地方。

二條城裡鑲的家徽有兩種。 一種是菊花，另外一種是葵花。 一般而言，一座城堡只鑲有一種 家徽。二條城會有兩種家徽，這 是因為二條城原本屬於德川家， 而德川家的家徽是葵花，但在日 本明治時代屬於朝廷，而天皇的 家徽是菊花。原本，二條城在歸 屬天皇時會卸下葵花，但有一處 屋樑上的德川家徽因為太高無法 卸下，而成為全二條城中，唯一 留下家徽比皇徽高的特例。這種 罕見的情形可以在二之丸御殿最 高的三角屋頂上看到。
